# 介護シフト管理システム - デモ版

## 概要

このデモ版は、ポートフォリオ展示用に作成された介護シフト管理システムのサンプルです。

> ⚠️ **注意**: すべてのデータは架空のものです。実在する人物・団体とは一切関係ありません。

## 機能一覧

| 機能 | 説明 | ファイル |
|------|------|----------|
| サービス報告 | 訪問介護のサービス記録 | `service-report/index.html` |
| 月間シフト | 月単位のシフト管理 | `monthly-shift/index.html` |
| 利用者一覧 | 利用者情報の管理 | `user-list/index.html` |
| 社員管理 | 職員情報の管理 | `employee-list/index.html` |

## 起動方法

### ローカルで確認

1. このフォルダを任意のWebサーバーで配信
2. `index.html` をブラウザで開く

**VSCode Live Server の場合:**
```
1. VSCode で portfolio-demo フォルダを開く
2. index.html を右クリック → "Open with Live Server"
```

**Python 簡易サーバーの場合:**
```bash
cd portfolio-demo
python -m http.server 8000
# ブラウザで http://localhost:8000 にアクセス
```

## デモアカウント

**デモ版ではログインは不要です。**

## PII除去について

このデモ版では以下のPII（個人情報）を完全に除去しています：

- Firebase接続設定（apiKey, projectId等）
- Google Apps Script URL
- 実在する人物・事業所名
- 実在する住所・電話番号
